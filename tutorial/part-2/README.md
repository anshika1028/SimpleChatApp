Multi-Room Chat
=============
This is the code for part 2 of the chat application tutorial.<br/>
<br/>
Install dependencies:
```
npm install
```
<br/>
Run the server:
```
node server.js
```
<br/>
Then open the following address in your browser:
```
http://localhost:4040
```
